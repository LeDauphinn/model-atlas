Big picture

What belongs where?

These names are related, but they are not interchangeable. Keeping the hierarchy straight makes the rest of ML much easier.

Machine Learning

The broad idea: learn patterns from data instead of hard-coding every rule.

Deep Learning

A subset of machine learning that uses many stacked layers of learned weights.

NLP

Natural language processing: applying ML to text, speech, and human language tasks.

LLM

A large language model: a deep learning model trained on a huge amount of text to predict tokens.

GPT

A specific style of transformer-based LLM that generates text autoregressively, one token at a time.

Supervised Input + label

Learn from examples with known answers.

Unsupervised Input only

Discover structure such as clusters or compressed features.

Self-supervised Create labels from data

Hide part of the input and learn to predict it.

Reinforcement Action + reward

Learn by trial, feedback, and long-term consequences.

Mental model

The six words you need everywhere

These ideas reappear in almost every ML project, from linear regression to GPT training.

Data

The examples you learn from. Better data often beats fancier models.

Features

The useful signals inside the data. Deep learning often learns features automatically.

Model

A parameterized function that turns inputs into predictions.

Loss

A number that says how wrong the model currently is.

Optimizer

The procedure that updates parameters to reduce loss.

Generalization

The model should work on new examples, not just memorize training data.

Depth matters

Why deep learning changed the field

Classic ML often relies on hand-designed features. Deep learning learns both the features and the final predictor.

Classic ML pipeline

  • Humans decide which features to extract.
  • Works well on structured tables and smaller datasets.
  • Models include linear regression, trees, SVMs, and k-means.

Deep learning pipeline

  • Neural networks learn internal features automatically.
  • Thrives on images, audio, language, and very large datasets.
  • Models include CNNs, RNNs, transformers, diffusion models.
Perceptrons

Early neural models showed the core idea: weighted sums plus nonlinear decisions.

CNNs

Convolutions became dominant for computer vision by learning local visual patterns.

RNNs / LSTMs

Sequence models handled time and language but struggled with long-range memory.

Transformers

Attention let every token look at every other token, which scaled better for language.

Foundation models

Huge self-supervised models became reusable bases for many downstream tasks.

How GPT-style models work

A GPT is a transformer trained to predict the next token. During inference, it keeps repeating that process: read context, score possible next tokens, sample one, append it, continue.

1

Pretraining

Read huge text corpora and learn next-token prediction.

2

Representation learning

Internal layers learn syntax, facts, style, and reusable abstractions.

3

Instruction tuning

Extra training teaches the model to follow tasks more directly.

4

Sampling at inference

Temperature and decoding choices shape how deterministic or creative the output feels.
